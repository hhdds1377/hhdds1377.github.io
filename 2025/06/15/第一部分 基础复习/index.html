值异操函作,函异模上循,类方生占.

  5. 作用域 - 2

    global nonlocal

  6. 条件 - 3

    if elif else

  7. 异常 - 6

    try except else* finally raise assert

  8. 模块 - 3

    import from as

    for while break continue

2.3 数值的函数

2.3.1 数值的工厂函数

  1. bool()

    bool()

    返回obj对象的布尔值.

  2. int()

    int(obj,base=<进制>)

    返回一个字符串或数值的整型,

    <进制>是可选的进制参数,类型是数值,默认是10.

  3. long()

    long(obj,base=<进制>)

    返回一个字符串或数值的长整形,

    <进制>是可选的进制参数,类型是数值,默认是10.

  4. float()

    float(obj)

    返回一个字符串或数值的浮点型.

  5. complex()

    complex(str) or complex(real,imag=0.0)

    放回一个字符串的复数型,或者根据给定的实数和一个可选的虚数部分,生成一个复数型.

2.3.2 数值的功能函数

  1. abs()

    abs(num)

    放回num的绝对值.

  2. round()

    round(flt,ndig)

    接收一个浮点型flt并对其四舍五入,保存小数点后ndig位小数,默认保存小数点后0位小数.

2.3.3 仅适用整型的函数

  1. hex()

    hex(num)

    返回num的十六进制的字符串类型.

  2. oct()

    oct(num)

    返回num的八进制的字符串类型.

  3. chr()

    chr(num)

    将ASCII码转换成ASCII字符,范围只能是0<=num<=255

  4. ord()

    ord(num)

    接收一个ASCII或Unicode字符,返回相应的ASCII码或Unicode码.

  5. unichr()

    unichr(num)

    接收Unicode码,返回对应的Unicode字符.

2.4 相关模块

模块 介绍
decimal 十进制浮点运算
math/cmath 标准C库数学运算函数,常规数学运算在math模块,复数运算在cmath模块
random 多种伪随机生成器

2.4.1 模块:random

  1. randint()

    两个整型参数,返回两者之间随机整型

  2. randrange()

    它接受和range()函数一样的参数,随机返回range()结果的一项

  3. uniform()

    几乎和randint()一样,不过它返回的是两者之间的一个浮点型

  4. random()

    类似uniform(),只不过下限恒等于0.0,上限恒等于1.0

  5. choice()

    返回给定序列的一个元素.
